# weztermconfig

Install 'Jetbrains Mono Nerd Font', to use this config

```shell
brew install --cask font-jetbrains-mono-nerd-font
```

## Setup

Clone the repo and symlink the config to your home directory:

```shell
git clone <repo-url> ~/work/knowledge/weztermconfig
ln -sf ~/work/knowledge/weztermconfig/.wezterm.lua ~/.wezterm.lua
```

After that, `git pull` will automatically apply any changes.
